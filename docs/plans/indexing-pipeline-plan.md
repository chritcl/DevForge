## 验收标准

- 三种语言（TypeScript、Rust、Python）能够稳定提取主要符号
- 文件修改后可以增量更新
- 文件删除后不会残留搜索结果
- 应用崩溃后索引任务可以恢复
- 当前 Active Index 不会因重建失败损坏
- 中型仓库（50,000 文件）首次索引性能达到内部目标
- Fixture 索引回归测试稳定
