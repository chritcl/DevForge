# 阶段八：发布、安全与产品化

## 目标

将工程原型转化为可以长期升级和分发的桌面产品。

## 功能范围

- Stable、Beta、Nightly 渠道
- Windows Code Signing
- Tauri Updater Signing
- 数据库升级备份
- 索引后台迁移
- 更新安全重启
- 崩溃恢复
- 安全模式
- 诊断中心
- 脱敏诊断包
- SBOM
- 依赖审计
- 性能回归
- 安装与升级测试
- 隐私说明
- 本地数据管理页面

## 退出条件

- 上一个正式版本可以直接升级。
- 升级不会删除旧索引，除非新索引成功。
- 更新期间不会中断关键写操作。
- 安装包和更新包签名验证通过。
- 诊断包默认不包含源码和 Prompt。
- 连续崩溃可以进入安全模式。
- Release CI 具备完整可重复流程。
